Esta fué nuestra pequeña aportación en la global game jam en 2022

¡Jugarás como dos mejores amigos, un vampiro y un ajo! Estos dos deberán cooperar para acabar con la tiranía de los malvados vampiros.

El vampiro puede moverse más suave que el ajo, pudiendo saltar más alto e incluso hacer saltos dobles. 
Además, puede activar palancas para que el ajo pueda atravesar obstáculos. El ajo es más lento pero es el único que puede derrotar a otros vampiros por lo que es clave en esta búsqueda.
